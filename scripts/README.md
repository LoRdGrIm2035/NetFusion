# NetFusion Scripts

This folder contains non-entrypoint helper scripts.

- `diagnostics/` contains validation and profiling helpers.
- `repair/` contains adapter repair helpers for known local recovery cases.

Operational entrypoints remain at the repository root:

- `NetFusion-START.bat`
- `NetFusion-STOP.bat`
- `NetFusion-SAFE.bat`
- `Setup-NetFusion.ps1`
- `Install-Service.ps1`
- `test-combined-speed.ps1`
